# Matcha Maker

## Description
A cute gamified matcha-making website where users choose ingredients, whisk matcha, reveal a final drink, unlock rewards, and save recipes.

## Project Overview
Matcha Maker is a cozy, interactive web experience built as a purely static website. It allows users to simulate the process of making matcha, featuring a whisking mini-game that calculates frothiness and reveals a uniquely generated matcha drink result. It also includes a gamified rewards roadmap and a collection to save and filter favorite recipes.

## Main Features
- **Gamified matcha-making flow:** Tap to whisk and perfectly froth your matcha.
- **Matcha result reveal:** See your personalized matcha drink with score and taste profile.
- **Rewards milestone roadmap:** Track your journey and unlock sweet milestones.
- **Saved recipe cards:** Collect, favorite, and browse through your saved matcha creations.
- **Cute pastel design system:** A cozy cafe aesthetic with rounded corners and pill-shaped buttons.
- **Responsive static website:** Works smoothly on desktop, tablet, and mobile browsers.

## How to Open
This is a clean static website with no build tools or frameworks required.
- **Directly:** Open `index.html` directly in any modern web browser.
- **Live Server:** Open the `matcha-maker` folder in your code editor (e.g., Cursor, VS Code) and use the "Live Server" extension to serve the files locally.

## Design Style
The UI features a cozy, pastel cafe aesthetic:
- **Shapes:** Rounded corners (`border-radius: 24px`), pill-shaped buttons (`9999px`), soft shadows.
- **Typography:** Friendly and rounded sans-serif font (Nunito).
- **Interactions:** Smooth scrolling, scale-on-click interactions for whisking, pop-in animations for badges.
